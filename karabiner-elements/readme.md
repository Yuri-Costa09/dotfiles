# What is Karabiner-Elements?

Since I use Aerospace (My tiling window manager), and also my split keyboard (Silakka-64 for now), I prefer to remap the capslock to be the key to use the window manager.

Basically, this is what these configs do.

# Step By Step

1. Go to 'Complex Modifications'
2. Press 'Add Your Own Rule'
3. Paste the code from ```./karabiner-elements.json```
